Olá pessoal do Luizalabs, sejam bem-vindos !

Facebook bot pronto para testes, Abaixo alguns comentários sobre o Setup.

As versões utilizadas pela app foram:
  • Django 1.9
  • Python 2.7
  • Bootstrap 4

Utilizei VM da Cloud9 com Ubuntu 14.04.5 LTS

Duas apps em Django/Python para resolver o exercício

Uma app web (luizalabs_web) responsável pela criação da página base em HTML, CSS e Javascript.

App (luizalabs_bot) engine de conexão as API's do Facebook, Google e chamadas JSON.

O código está bem simples, e espero ter alcançado os objetivos do Exercício.

Para dar início, por favor, seguir os passos abaixo:

  1. Sair do facebook caso esteja logado.
  2. Selecione o widget de mensagem e envie uma mensagem como "Boa tarde, Olá, etc..."
  3. Quando solicitado, por favor, logar com o usuário teste no facebook:
    • usuário: apptester2017.br@gmail.com
    • senha: apptester2017br

    Importante: Como a app ficou muito customizada, não solicitei publicação dela pelo Facebook. Portanto somente usuários testers pode usar. Caso precisem me informem um usuário do facebook e eu posso adicionar, sem problemas.

  4. Abra o facebook usando as credenciais do usuário teste e BOAS COMPRAS !! ;)
  5. Ao terminar suas consultas, por favor, finalize dizendo obrigado !
